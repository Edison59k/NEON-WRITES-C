Authentication & Data Flow

Action Trigger Data Stored Redirect
Sign Up User clicks "Sign Up" button on index.html neon_writers_user (JSON)
neon_writers_logged_in = true 		mainpage.html
Log In User clicks "Log In" button on index.html neon_writers_user (JSON)
neon_writers_logged_in = true 		mainpage.html
Access Protected Page User tries to visit mainpage.html or profile.html Check localStorage for logged_in flag If not logged in → index.html
Update Profile User submits profile form on profile.html Updated neon_writers_user JSON Page reload with success message
Logout User clicks logout on any protected page Clear localStorage (remove both keys) index.html after 1500ms

User Journey Examples

New User Journey

  1. Land on index.html
  2. Click "Start Earning Now" or "Sign Up" button
  3. Enter credentials (First Name, Last Name, Email, Phone)
  4. System stores data in localStorage
  5. Automatically redirected to mainpage.html
  6. View personalized dashboard with welcome message
  7. Click "My Profile" in sidebar
  8. View and edit profile information
  9. Click "Logout" to return to index.html

Returning User Journey

  1. Land on index.html
  2. Navbar buttons change to "Dashboard" and "My Profile"
  3. Click "Dashboard" to go to mainpage.html
  4. View dashboard with saved user data
  5. Access other pages via sidebar (Surveys, Payments, etc.)
  6. Click logout to clear session and return to home
